# Weather Studio

Weather Studio is a ReactJS-based weather application that allows users to check the current weather and 5-day forecast for a specific city.

## Getting Started

### Prerequisites

Before running the project, you need to create an account on OpenWeatherMap and obtain an API key.

1. Create an account on [OpenWeatherMap](https://openweathermap.org/) and get your API key.
2. Create a `.env` file in the root of your project and add the following line:

   ```env
   REACT_APP_APIKEY=your_openweathermap_api_key
## Installation

1. **Clone the repository:**

    Open your Git Bash and run the following command:

    ```bash
    git clone https://github.com/your-username/weather-studio.git
    ```

2. **Change into the project directory:**

    ```bash
    cd weather-studio
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

## Run the Application

Start the project locally:

```bash
npm start
